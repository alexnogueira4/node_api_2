# NODE Rest full API

### About
A Node REST API with a CRUD of users

### Features
+ Restfull
+ MongoDB
+ Tests with Mocha
+ Express
+ Typescript
+ Docker
+ Passport
+ Data Validation with Joi
+ JSON Web Token - JWT

### How to Install
````
npm i
````

### How to run
````
docker-compose up --build
````

### How to start
````
npm start
````

### How to test
The tests for each version of the API can be found in ````./test````, and each route has its own test file.

To run the tests:

````
npm test
````
